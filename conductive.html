Texting thumb experiment

This experiment had two purposes. First of all we wanted to use technology to help people become more aware of the physological happenings/changes that the smartphone creates. Second of all we were eager to experiment with conbinations of material. The combination of the more advanced conductive thread and arduino mini with the soft and xx fabric seemed to be an exiting combination that still intruiqes me.
We had to calculate the threashold for the amount of streatch that the different materials reacted to to be able create an output for the user to understand. We ended up using vibration because we wanted the output to be stricly personal and only for the user to sense.
