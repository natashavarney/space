NearBy

NearBy is a prototype for an application. The applications vision is create a platform where all food coupons like Tipster ect. are gathered to create a easier way to find the food coupons near you. The application uses geolocation to locate the restaurents that are currently offering a deal.
We had to calculate the threashold for the amount of streatch that the different materials reacted to to be able create an output for the user to understand. We ended up using vibration because we wanted the output to be stricly personal and only for the user to sense.
